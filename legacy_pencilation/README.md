# Pencilation 🎨
## Artistic Portrait Commission & Booking System

Capture the essence of your memories with **Pencilation**. A premium, high-fidelity booking platform designed for independent portrait artists to showcase their work and streamline commissions.

---

### ✨ Key Features

- **Dynamic Navigation Dock**: A sleek, glassmorphism-inspired floating navigation dock for effortless site discovery.
- **Interactive Portfolio**: Smoothly animated gallery showcasing realism, digital fauvism, and charcoal studies.
- **Real-time Booking System**: Integrated commission form with dynamic pricing, medium selection, and custom size options.
- **Interactive Location Picker**: Built-in Leaflet.js map for scheduling meet-ups with precision.
- **Secure Artist Dashboard**: Private portal for managing bookings, gallery items, and system settings.

### 🛠️ Technology Stack

- **Frontend**: HTML5, Vanilla JavaScript, CSS3
- **Styling**: Tailwind CSS (Utility-first) & Custom Premium Animations
- **Icons**: Lucide-Icons (Clean & Modern)
- **Maps**: Leaflet.js Integration
- **Backend**: PHP-compatible architecture for seamless data processing

---

### 🎨 Design Philosophy

Pencilation is built with a **Fauvism-inspired** aesthetic, combining bold typography with a minimal, "paper-like" background for a truly artistic feel. Every interaction is designed with micro-animations and smooth transitions to provide a premium user experience.

---

### 🚀 Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Ambatukam2001/Portrait-Commission-System.git
   ```
2. **Setup Local Environment**:
   - Host on a PHP-capable server (e.g., XAMPP, InfinityFree).
   - Import the provided `database.sql` to your MySQL instance.
   - Configure `js/config.js` for your API endpoints.
3. **Run**: Open `index.html` via your local server and start commissioning!
